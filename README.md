# Superstore Data Analysis

## Key Insight
High discount levels (>30%) are the primary driver of loss making transactions, with the Furniture category being the most affected due to consistently higher discount application.

## Objective
To identify key factors influencing profitability in a retail superstore dataset.

## Dataset Overview
The dataset contains 9,994 order line records. Each row represents one product within an order.

## Tools Used
- SQL
- Power BI
- Microsoft Excel

## Key Findings
- Standard Class generates the highest total profit but lowest profit per order
- Discounts above 30% are strongly associated with losses
- Furniture is the least profitable and most heavily discounted category
